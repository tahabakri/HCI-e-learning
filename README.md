# HCI E-Learning Student Dashboard

## Description

This project is a student dashboard for an e-learning platform, designed to demonstrate Human-Computer Interaction (HCI) principles. It provides an intuitive interface for students to access courses, view timelines, manage notifications, and interact with professors.

## Features

- **Dashboard Overview**: Quick access to recently accessed courses
- **Subject List**: Browse available subjects
- **Timeline Calendar**: View upcoming assignments, deadlines, and events with filtering options
- **Notifications**: Real-time notifications with badge counts
- **Profile Section**: Access to email, profile view, and professor contacts
- **Dark Mode Toggle**: Switch between light and dark themes for better usability
- **Responsive Design**: Optimized for various screen sizes

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/tahabakri/HCI-e-learning.git
   ```

2. Navigate to the project directory:

   ```bash
   cd HCI-e-learning
   ```

3. Open `index.html` in your web browser.

## HCI Principles Demonstrated

- **Usability**: Intuitive navigation and clear information hierarchy
- **Accessibility**: Responsive design and keyboard navigation support
- **User-Centered Design**: Features based on student needs (courses, deadlines, communication)
- **Feedback**: Toast notifications and visual indicators
- **Consistency**: Uniform styling and interaction patterns
